# Active Directory Home Lab Portfolio

## Overview
Built a complete Active Directory environment from scratch using VirtualBox,
demonstrating real-world IT support skills across 12 helpdesk scenarios.

## Lab Environment
| Component | Details |
|-----------|---------|
| Domain | lab.local |
| DC-01 | Windows Server 2025 — 192.168.10.10 |
| CLIENT-01 | Windows 11 Pro — 192.168.10.20 |
| Hypervisor | VirtualBox 7.x |
| Network | Internal Network (adlab) + NAT |

## Skills Demonstrated
- Active Directory administration (users, groups, OUs)
- Group Policy management (GPO creation, linking, testing)
- PowerShell automation (bulk user creation, auditing)
- User lifecycle management (onboarding, offboarding, transfers)
- Security policy implementation (PSO, password policies)
- Software deployment via GPO
- AD security auditing
- DNS configuration and troubleshooting
